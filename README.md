To-Run

FOLLOW THE BELOW STEPS ONE BY ONE

1) Create the .venv by running the below script
      python -m venv .venv

2) Active the .venv by the following script
     .venv/scripts/activate

3) To install all dependencies
     pip install -r requirements.txt

4) To run the
     python app.py

   You will see the [ http://localhost:5](http://127.0.0.1:5000)
